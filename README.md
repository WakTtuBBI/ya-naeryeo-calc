# 공익교대 — 지하철공익 월급계산기

지하철에서 근무하는 **사회복무요원(지하철공익)**을 위한 월급·근무 계산기입니다.
2교대(주야비휴) 교대근무를 달력에 그대로 입력하고, 계급·식비·교통비·야간 2일 처리까지 반영된 월급을 자동으로 계산합니다.

> ⚠️ **비공식 도구입니다.** 이 앱은 기존 사회복무요원 월급계산 앱 **「공익인간」**(ROIS Studio)에서 영감을 받아, 그 앱이 지원하지 않는 **지하철공익 교대근무**를 계산하기 위해 별도로 새로 제작한 비공식 보조 도구입니다. 공익인간과는 관련이 없으며, 코드를 가져오지 않고 처음부터 직접 작성했습니다. 계산 결과에 법적 효력은 없으며 실제 지급액은 복무지·규정에 따라 다를 수 있습니다.

## 왜 만들었나

공익인간 같은 기존 앱은 근무일을 **평일/공휴일**로만 나눠 계산합니다.
하지만 지하철공익은 **주간·야간·비번·휴무**가 도는 교대근무라, 쉬는 날이 주말과 무관하게 정해지고 **야간은 2일 근무(숙직)로 처리**됩니다.
그래서 기존 앱으로는 정확한 월급 계산이 어려웠고, 이 점을 개선하려고 만들었습니다.

근무계획서가 엑셀표로만 나와서 불편했는데, 이 앱에 입력해두면 **내가 언제 일하고 쉬는지 달력으로 한눈에** 볼 수 있는 것도 장점입니다.

## 주요 기능

- **교대 루틴 설정** — 3조2교대 / 4조2교대(주야비휴) / 통상근무 프리셋 + 사이클 직접 편집, 사이클 시작일 지정
- **달력 입력** — 날짜별 주간/야간/비번/휴무 + 휴가 입력, 휴무는 직접 지정(야간 로테이션 안 밀림), 야간 입력 시 다음날 자동 비번
- **월급 자동 계산** — 소집일 기준 계급 자동 진급(이병 2개월·매월 1일·일할계산), 기본급 + 식비 + 교통비, **야간 2일치** 반영, 연도별 식비 단가(2025년 7,000원 / 2026년~ 9,000원)
- **휴가 관리** — 1·2년차 연가, 병가, 특별/청원/공가. 잔여 자동 집계(반가 0.5일), 한도 초과 입력 차단, 날짜별 사유 메모
- **휴일 한도** — 그 달 주말+공휴일 수만큼만 휴무 입력 가능
- **실시간 복무율** — 현재 시각 기준 % 게이지(소수점 자리수 선택), D-day, 근무일
- **군적금 계산기** — 회차별 납입액·이자·매칭지원금 추정
- **백업/복구** — JSON 내보내기/불러오기

## 사용법

1. 처음 열면 **소집일·소집해제일** 입력 (소집해제일은 1년 9개월 기준 자동, 다르면 수정)
2. **루틴 탭** — 본인 근무 형태와 사이클 시작일 설정
3. **설정 탭** — 식비·교통비, 연가·병가·특휴 일수 등 본인 값 입력
4. **달력 탭** — 실제 근무표대로 휴무·휴가 입력 → 월급 자동 계산 확인

> 데이터는 **각자 기기 브라우저(localStorage)에만 저장**되며 서버로 전송되지 않습니다. 브라우저 데이터를 지우면 사라지니 **설정 → 백업·복구**로 내보내기를 권장합니다.
